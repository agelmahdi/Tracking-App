# TrackingApp
This is a practical project for Professional Android Developers.

# Covered Skills
1. Real-like coding with Kotlin.
2. MVVM Architecture.
3. Repository pattern.
4. Foreground Location service.
5. Kotlin Coroutines.
6. LiveData Streaming.
7. Room database.
8. Dependancy injection dagger-hilt.
9. ViewBinding.
